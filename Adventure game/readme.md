# The Lost World Cup Trophy

## Overview

The Lost World Cup Trophy is an interactive text-based adventure game built in Python.

Just hours before the opening ceremony of the 2026 FIFA World Cup at MetLife Stadium, the World Cup Trophy is stolen. As FIFA's top football detective, the player must travel across multiple countries, collect clues, make important decisions, and recover the trophy before the tournament begins.

## Features

* Interactive story-based gameplay
* Multiple countries to investigate
* User-driven choices
* Score tracking system
* Multiple endings based on player decisions
* Football-themed adventure

## Countries Visited

* Argentina
* Brazil
* Portugal
* France
* MetLife Stadium (Final Mission)

## Concepts Used

This project demonstrates the following Python concepts:

* Variables
* User Input (`input()`)
* Conditional Statements (`if`, `elif`, `else`)
* Functions
* Global Variables
* String Formatting
* Program Flow Control

## How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python adventure_game.py
```

## Gameplay Objective

Recover the stolen FIFA World Cup Trophy by making the correct decisions throughout the investigation.

Different choices lead to different outcomes, including:

*  Perfect Ending
*  Good Ending
*  Bad Ending
