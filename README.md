## Supplementary Task 1: Weather Alert System

### Description

A Python application that uses the OpenWeatherMap API to monitor weather conditions for a selected city.

If:

* Temperature exceeds 35°C, or
* Rain is predicted

the script automatically sends an email alert.

### Features

* OpenWeatherMap API integration
* Weather forecast retrieval
* Rain detection
* Temperature threshold monitoring
* Automated email notifications
* Environment variable support for API keys and credentials

### Technologies Used

* Python
* Requests
* OpenWeatherMap API
* SMTP Email
* python-dotenv
