Learning CSS

07 June 2026

After learning the basics of HTML, I started exploring CSS to make webpages visually appealing.

Initially, my webpages looked very plain because HTML focuses on structure rather than design. CSS helped me add colors, spacing, borders, and layouts to improve the appearance of my website.

While building my portfolio, I learned how to use selectors, classes, and IDs to target specific elements. I also experimented with hover effects, shadows, transitions, and rounded corners.

One of the most interesting concepts I learned was Flexbox. It allowed me to arrange project cards and skill cards side by side while keeping the layout organized.

Learning CSS showed me how design and usability play an important role in creating a good user experience.
