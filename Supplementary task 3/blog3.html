Figma design journey

04 June 2026

Apart from coding, I enjoy exploring UI/UX design using Figma. It helps me visualize ideas before implementing them in code.

One of the projects I designed was a task management application called Tide. The goal was to create a clean and user-friendly interface that helps students manage their daily tasks and deadlines.

I also worked on a trading application prototype called AIVESTOR. Through this project, I learned about dashboard layouts, color schemes, and designing intuitive user flows.

Figma taught me the importance of consistency in design. Small details such as spacing, typography, and alignment can greatly improve the overall user experience.

Combining design skills with web development allows me to build applications that are both functional and visually appealing.
