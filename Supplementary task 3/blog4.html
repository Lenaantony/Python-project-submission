Building my portfolio

10 June 2026

Creating my personal portfolio website has been one of the most rewarding learning experiences in my web development journey.

I started with a simple webpage containing only headings and paragraphs. Gradually, I learned how to organize content using sections, navigation bars, and reusable cards.

As I progressed, I added Flexbox layouts, CSS Grid, hover animations, project filtering using JavaScript, and embedded my Figma designs directly into the website.

Through this project, I gained practical experience with HTML, CSS, and JavaScript while understanding how these technologies work together to create interactive websites.

This portfolio not only showcases my projects and skills but also reflects my growth as a learner and aspiring web developer.
